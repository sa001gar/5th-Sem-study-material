## About the Files

- **Software Engineering**: 56 individual topic files with mermaid diagrams, tables, code examples, and inter-topic navigation links. Each topic preserves original study content with enhanced formatting.
- **Notes (`notes.md`)**: Contains the structured syllabus, detailed theoretical explanations, examples, and markdown-based diagrams for each unit. Each notes file includes a clickable index at the top for easy navigation to specific units and sub-topics.
- **Practice Questions (`practice_questions.md`)**: Contains categorized questions (2-mark, 5-mark, and 10-mark formats) organized by unit, complete with detailed model answers. It also features a clickable index for quick reference.
